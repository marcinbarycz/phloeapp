PHLOE - SEATING APP (offline lokalna wersja)

Kroki uruchomienia (MacOS):
1. Wymagane:
   - Node.js: https://nodejs.org
   - Terminal

2. Rozpakuj folder:
   cd phloe-seating-app

3. Zainstaluj zależności:
   npm install

4. Uruchom lokalnie:
   npm run dev

5. Otwórz w przeglądarce:
   http://localhost:3000
