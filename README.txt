PHLOE - WERSJA 2: Logowanie, rejestracja, profil plannera

Jak uruchomić (na MacOS):
1. Wymagane:
   - Node.js: https://nodejs.org
   - Terminal

2. Rozpakuj folder:
   cd phloe-seating-app-v2

3. Zainstaluj zależności:
   npm install

4. Uruchom lokalnie:
   npm run dev

5. Otwórz przeglądarkę:
   http://localhost:3000
